## <a name="about">🤖 サイトの概要</a>

収納ラックの個数と段数・素材・色を自由に選べる 3D コンフィグレーターです。<br />
収納ラックのモデルは Blender で作成し、Three.js でオプションの
カスタマイズができるようになっています。<br />
また、個数や段数の選択別に値段も増減するよう制作しました。<br />
<br />
制作時期：2024年5月頃

## <a name="more">📙 あとがき</a>

モデルは1ボックスのみ作成し、個数や段数の選択に応じてモデルも増減して表示されるよう制作を行いました。<br />
ボックスが増えた場合の位置調節の計算が難しく、若干のズレなどが生じる部分もありましたが、<br />
最終的に違和感がない場所に設置することができました。<br />
<br />
今後も Three.js に関する知識を深めていけたらと思います。<br />
最後までお読みいただきありがとうございました！
